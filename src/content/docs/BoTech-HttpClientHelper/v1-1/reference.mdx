## Methods
### Http-Get

<details>
    <summary>HttpGetFile</summary>
    <Code code="public async Task<RequestResult<dynamic>> HttpGetFile(string url, string fileName)" lang="c#" />
    + General: Performs a GET request to the given endpoint (baseUrl + url). This Method also saves the file to a specific location defined in fileName.
    + Param (fileName): The full path of a file to overwrite or create.
    + Param (url): The endpoint url
    + Returns: The request result.
</details>
<details>
    <summary>HttpGetJsonObject</summary>
    <Code code="public async Task<RequestResult<T>> HttpGetJsonObject<T>(string url)" lang="c#" />
    + General: Performs a GET request to the given endpoint (baseUrl + url)
    + Param (url): The endpoint url
    + Returns: The request result with parsed json data object
</details>
<details>
    <summary>HttpGetString</summary>
    <Code code="public async Task<RequestResult<string>> HttpGetString(string url)" lang="c#" />
    + General: Sends a request to _baseUrl + url and returns the string returned by that method.
    + Param (url): The endpoint url
    + Returns: The returned string from the api and the request result.
</details>
<details>
    <summary>HttpGet</summary>
    <Code code="public async Task<RequestResult<dynamic>> HttpGet(string url)" lang="c#" />
    + General: Performs a GET request to the given endpoint (baseUrl + url)
    + Param (url): The endpoint url
    + Returns: The request result with no parsed json data.
</details>
